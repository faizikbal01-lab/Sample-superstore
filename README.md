# Sample-superstore
A comprehensive Excel Mastery project covering 12 core sections: from basic arithmetic and data cleaning to advanced Pivot Tables, LOOKUP functions, and VBA Macros. Includes hands-on tasks for financial modeling, data visualization, and automation.
This repository contains a comprehensive set of Excel workbooks and documentation covering a full spectrum of spreadsheet skills. The project is divided into 12 strategic sections designed to demonstrate proficiency in data manipulation, financial analysis, and process automation.

Section 1-3: Fundamentals & Formatting
Basic Arithmetic: Implementation of SUM, MAX, MIN, SMALL, LARGE, and MEDIAN.

Transposition: Comparing manual transpose vs. the dynamic TRANSPOSE formula.

Counting Functions: Usage of COUNT, COUNTA, and COUNTBLANK for data auditing.

Growth Analysis: Calculating Year-over-Year (YoY) growth and Compound Annual Growth Rate (CAGR) using the POWER formula.

Professional Formatting: Application of Conditional Formatting, Custom Number formats, and Format Painter for data visualization.

Section 4-6: Data Cleaning & Logic
Date Engineering: Calculating durations with timestamps and using DATEDIF, TODAY, and DATE functions.

Logical Operations: Complex nesting of IF, AND, OR, and IFERROR.

Data Scrubbing: Cleaning datasets using TRIM, PROPER, CLEAN, TEXT TO COLUMNS, and Duplicate Removal.

Advanced Extraction: Substring manipulation using SEARCH, FIND, MID, LEFT, and RIGHT.

Section 7-9: Advanced Data Analysis
Pivot Tables: Mastery of grouping (Dates/Categories), Slicers, and Conditional Formatting within Pivots.

Lookup Formulas: Vertical and Horizontal lookups (VLOOKUP, HLOOKUP) with exact and approximate matches.

Conditional Aggregation: Advanced criteria-based counting and summing using SUMIFS, COUNTIFS, and AVERAGEIFS.

Section 10-12: Reporting & Automation
Report Consolidation: Using the Consolidate feature to aggregate data from multiple sheets and the SUBTOTAL function for filtered lists.

VBA Macros: Introduction to automation, including:

Enabling Developer environments.

Recording and editing Macros.

Assigning Macros to Buttons and Keyboard Shortcuts.
